nifti: the standard toolbox for reading NIFTI files
aux_files: includes files needed for the nifti generation. The transformation matrix as generated by FSL (4mm_to_0.5mm.mat), mni152bet.nii for the high resolution image and MNI152_T1_4mm.nii to copy the header

vector2pic: transforms the vector of importances to 3-D image to be stores in the nifti file.
